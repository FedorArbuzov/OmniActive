# Идеи фич для OmniActive

## Текущий функционал

- **Авторизация** — вход/регистрация, реферальные коды
- **Профиль** — рост, вес, возраст, планы тренировок, достижения, рефералы, OTA-обновления
- **Тренировки** — зал, баскетбол, футбол, хоккей — создание и выполнение
- **Активность** — шаги, PAL (фиксированный/ежедневный), режимы учёта
- **Калории** — блюда, дневник питания, копирование рациона дня
- **Упражнения** — быстрый ввод (отжимания, подтягивания и т.д.), статистика
- **Планы тренировок** — кастомные расписания, обмен по коду
- **Бэкенд** — полноценный API, синхронизация, достижения

---

## Предложения новых фич

### Быстрые в реализации (1–2 дня)

1. **Таймер для упражнений** — для планки и упражнений на время: обратный отсчёт, звук/вибрация в конце.
2. **Поиск по блюдам** — при большом количестве блюд удобный поиск по названию.
3. **Расчёт суточной нормы калорий** — использовать рост/вес/возраст из профиля + коэффициент активности для BMR и нормы калорий.
4. **График веса** — экран с графиком изменения веса по датам (данные уже есть в профиле).
5. **Дашборд на главной** — краткая сводка «сегодня: X ккал / Y шагов / Z тренировок».

### Средней сложности (3–7 дней)

6. **Push-уведомления** — напоминания «пора записать калории», «время тренировки по плану», «N дней без активности».
7. **Экспорт данных** — выгрузка калорий и тренировок в CSV/PDF, backup.
8. **Сканер штрихкодов** — добавление продуктов по штрихкоду (если найдутся API баз продуктов).
9. **Прогресс в тренировках** — история рабочих весов и повторений по упражнениям, графики прогресса.
10. **Самочувствие после тренировки** — оценка настроения/усталости после тренировки для анализа.

### Крупные фичи (1–2 недели)

11. **Интеграция с Google Fit / Apple Health** — автоимпорт шагов и активности.
12. **Социальный функционал** — друзья, лидерборды, челленджи (у вас уже есть рефералы — можно расширить).
13. **AI-рекомендации** — подсказки на основе целей и статистики (питание, тренировки).
14. **Онбординг и цели** — при первом запуске: цель (похудеть/набрать/поддержка), расчёт калорий и предложение планов.
15. **Темы оформления** — несколько цветовых схем (сейчас есть light/dark, можно добавить варианты).

### Улучшения UX

16. **Swipe-действия** — удаление/редактирование записей свайпом в списках.
17. **Календарь активности** — визуализация: какие дни были активными, с тренировками, в рамках калорий.
18. **Быстрое копирование блюд** — «вчера съел то же» — одним тапом скопировать рацион.
19. **Прогресс к достижениям** — показывать «100 подтягиваний: 67/100» и т.п.
20. **Офлайн-режим** — явный индикатор офлайна и очередь синхронизации при восстановлении связи.
